## âš™ï¸ Installation & Setup  

1. **Clone the Repository**
   ```bash
   git clone https://github.com/<your-username>/cashless-village.git
   cd cashless-village
   ````

2. **Install Dependencies**

   ```bash
   npm install
   # or
   yarn install
   ```

3. **Run the Development Server**

   ```bash
   npm start
   # or
   yarn start
   ```

   The app will run on [http://localhost:3000](http://localhost:3000).

4. **Build for Production**

   ```bash
   npm run build
   ```

5. **Deploy**
   You can deploy it easily on:

   * **GitHub Pages** (via `gh-pages` package)
   * **Emergent Agent Preview**
   * **Vercel** or **Netlify**
